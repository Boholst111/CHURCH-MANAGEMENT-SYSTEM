# Quick Start Guide - Modern UI

## 5-Minute Tour of Your New Interface

Welcome! This quick guide will get you up and running in just 5 minutes.

---

## 1. Login (30 seconds)

**What you'll see:**
- Clean, modern login screen
- Church branding on the left (desktop)
- Simple login form on the right

**What to do:**
1. Enter your email and password
2. Check "Remember me" (optional)
3. Click "Sign In"

**Tip:** Forgot password? Click the link below the form.

---

## 2. Dashboard Overview (1 minute)

**What you'll see:**
- **Quick Stats** at the top (Members, Events, Visitors, Groups)
- **Finance Cards** showing offerings and expenses
- **Attendance Chart** with trends
- **Recent Activity** feed
- **Upcoming Events** list

**What to do:**
- Scan the quick stats for key numbers
- Check upcoming events
- Review recent activities

**Tip:** Everything updates automatically - no need to refresh!

## 4. Finding Information (1 minute)

### Quick Search
- Press `Ctrl/Cmd + K` anywhere
- Type what you're looking for
- Press Enter to go there

### Using Filters
On any list page (Members, Events, etc.):
1. Use the search bar at the top
2. Apply filters from dropdowns
3. Results update automatically

**Tip:** Combine search and filters for precise results!

---

## 5. Common Tasks (2 minutes)

### Adding a Member
1. Go to **Members**
2. Click **"+ Add Member"**
3. Fill in the form
4. Click **"Save"**

### Creating an Event
1. Go to **Events**
2. Click **"+ Create Event"**
3. Fill in details (title, date, location)
4. Click **"Create Event"**

### Recording an Offering
1. Go to **Finance** > **Offerings** tab
2. Click **"+ Record Offering"**
3. Enter date, type, and amount
4. Click **"Record"**

### Generating a Report
1. Go to **Reports**
2. Find the report you need
3. Click **"Generate"**
4. Select date range and format
5. Click **"Generate Report"**

**Tip:** Look for the **+ buttons** throughout the system for quick actions!

## Keyboard Shortcuts

Speed up your work with these shortcuts:

| Shortcut | Action |
|----------|--------|
| `?` | Show all shortcuts |
| `Ctrl/Cmd + K` | Quick search |
| `G then D` | Go to Dashboard |
| `G then M` | Go to Members |
| `G then E` | Go to Events |
| `G then F` | Go to Finance |
| `Esc` | Close modal |
| `N` | New item (on list pages) |

**Tip:** Press `?` anywhere to see all shortcuts!

## Dark Mode

Easy on the eyes in low light:

1. Click your **profile picture** (top-right)
2. Click **"Theme"**
3. Select **"Dark"**, **"Light"**, or **"Auto"**

**Auto mode** follows your device's theme.
